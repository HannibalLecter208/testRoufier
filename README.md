# testRoufier
*название:* Оценка фунуционального состояния сердечно_сосудистой системы с помощью теста Руфье
*цель проекта:* Исследование адаптационных возможностей сердца и уровня физического здоровья человека при физической нагрузке с использованием простогоБ но информативного нагрузочного теста - теста Руфье
*описание:* Проект основан на методике функциональной диагностикиБ разработанной Ж.Руфьк в середине 20 века. В рамках проекта участники выполняют стандартную физическую нагрузку. На основе частоты пульса, измеренной до, сразу после нагрузки и в период восстановления, расчитывается Индекс Руфье
